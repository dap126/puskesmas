# Sistem Informasi Pelayanan Puskesmas

## Deskripsi Proyek
Sistem Informasi Pelayanan Puskesmas adalah aplikasi berbasis web untuk mendigitalisasi operasional puskesmas, dari pendaftaran pasien hingga manajemen apotek. Sistem ini dibangun secara full-stack dari awal, di mana setiap anggota mengembangkan sisi backend dan frontend. Pengembangan dikolaborasikan menggunakan version control seperti Github agar proses terstruktur.

## Fitur Utama (Rencana)
- **Registrasi & Antrian Pasien:** Registrasi pasien online maupun offline. Pasien harus didata dan diberikan nomor antrian berdasarkan poli yang dituju.
- **Rekam Medis / Pemeriksaan:** Catatan riwayat kesehatan, diagnosis, dan tindakan dokter.
- **Manajemen Dokter & Poliklinik:** Sistem antrean Poli secara otomatis.
- **Farmasi & Inventaris Obat:** Pengelolaan inventaris obat dan resep.
- **Dashboard Admin:** Laporan statistik kunjungan dan layanan Puskesmas.

## Teknologi (Tech Stack)
- **Frontend:** HTML, CSS, JavaScript (Vue.js)
- **Backend:** Express (Node.js)
- **Database:** Mysql
